Home Piste Territorio Info

Le immagini sono state "prese in prestito" da google immagini.

Le informazioni riguardanti il territorio sono derivate da ricerca web, per la maggior parte wikipedia, e da conoscenze dell'autore.

Per contattare ed avere informazioni sull'autore si veda in fondo alla pagina.

Ogni parte di questo progetto e' accessibile attraverso la pagina github dedicata

Di particolare interesse possono risultare le seguenti funzionalita':

  • Funzione javascript per rimodulare le coordinate di una mappa html in base alla taglia dell'immagine
  • Funzioni javascript per l'interrogazione di file JSON e l'aggiunta di parti html derivate dai dati
  • Funzioni javascript [Node] per ottenere ed inviare dati a un database noSQL [Mongo]
  • Funzione javascript per mostrare cards in slideshow

Si segnala che parte del codice scritto e' stato derivato da esempi trovati online, per saperne di piu' si legga la descrizione del progetto nel repository github.
